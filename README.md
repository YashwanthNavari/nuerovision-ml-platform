## ✨ Key Features

- **⚡ Ultra-fast Build:** Powered by Vite for lightning-fast module updates.
- **🎨 Interactive Data Visualization:** Responsive graphs and charts using modern UI layers.
- **🤖 AI Integrated:** Connects seamlessly with Google Gemini AI Studio.

## 🚀 Get Started Locally

Follow these steps to set up the dashboard on your local machine:

### 1. Clone & Install
```bash
git clone https://github.com/YashwanthNavari/nuerovision-ml-platform.git
cd nuerovision-ml-platform
npm install
```

### 2. Configure Environment
Create a `.env.local` file and add your Gemini API Key:
```env
GEMINI_API_KEY=your-api-key-here
```
> **Note:** You can view or configure your app in AI Studio [**here**](https://ai.studio/apps/drive/18RbCX4YykN8mH-x3zJbN70EfI4p79x2I).

### 3. Run the App
```bash
npm run dev
```
